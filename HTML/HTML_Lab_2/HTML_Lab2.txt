1. ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร? (5 นาที)
ตอบ  <!doctype> มีหน้าที่บอก web browser ว่าเอกสารนี้เขียนโดยใช้ HTML version อะไร เพื่อให้มั่นใจว่า browser ทำงานอย่างถูกต้อง 


2. ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด? (3 นาที)
ตอบ title attribute
ตัวอย่าง: 
<a href=”” title=”tooltip_text”> Content </a>


3. แท็ก <b>, <i> ต่างจาก <strong>, <em> หรือไม่ อย่างไร? (5 นาที)
ตอบ	<b> คือ bold หรือตัวหนังสือแบบหนา ใช้สำหรับข้อความที่ต้องการดึงความสนใจ
	<i> คือ italic หรือตัวหนังสือเอียงขวาเล็กน้อย ใช้กับชื่อเรื่อง คำต่างประเทศ คำทับศัพท์ ความคิดของตัวละคร และอื่นๆ
	<strong> ใช้สำหรับข้อความสำคัญเป็นพิเศษ และมีความยาวที่ไม่มากจนเกินไป โดยจะแสดงผลแบบ bold
	<em> ใช้เพื่อดึงความสนใจเป็นพิเศษ โดยจะแสดงผลแบบ italic

4. หา tag ที่สามารถแสดงผลข้อความกลับหลังได้ เช่น โปรแกรมเมอร์ แสดงเป็น ร์อมเมรกแรปโ
ตอบ <bdo> tag 
ตัวอย่าง:
<!DOCTYPE html>
<html>
<body>

<h1>การแสดงผลข้อความกลับหลัง</h1>

<p>โปรแกรมเมอร์ </p>  
<p><bdo dir="rtl">โปรแกรมเมอร์ </bdo></p>  

</body>
</html>
